# 🚗 Car Rental Website (MERN Stack)

A full-stack Car Rental Web Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).
This platform allows users to browse cars, book rentals, and manage bookings, while the admin can manage cars and reservations.

<hr/>

# 📌 Features
👤 User Features

* User Registration & Login (Authentication)
* Browse available cars
* Book a car
* View booking history
* Contact support

# 🛠 Admin Features

 * Admin dashboard
 * Add new cars
 * Update car details
 * Delete cars
 * View all bookings
 * Manage users

 # 🧰 Tech Stack
#  Frontend
